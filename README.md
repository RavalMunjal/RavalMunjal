## 🧠 About Me

Hey! I'm **Munjal Raval** — a passionate Full Stack Developer from Gujarat, India. 🇮🇳

I build complete web applications using the **MERN stack**, focusing on both clean frontend experiences and solid backend systems. For me, development isn't just a skill — it's a way of thinking and solving real problems.

My approach is simple: **learn by doing**. Instead of only watching tutorials, I jump into projects, hit errors, debug them, and come out understanding things deeply. That's how I've built everything from emergency-response apps at hackathons to a secure digital locker for life-critical data.

I care about writing **clean, maintainable code** and building things that actually work in the real world — not just on localhost. I believe every project is an opportunity to grow, and every bug is a lesson in disguise.

Right now I'm focused on:
- 🔨 Building scalable full-stack applications
- 🧠 Sharpening my DSA and problem-solving skills
- 🏆 Competing in hackathons and collaborating with real teams
- 🌐 Working towards meaningful open source contributions

I'm actively looking for **internships and entry-level opportunities** where I can learn fast, contribute early, and grow alongside a great team.
